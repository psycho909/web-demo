# 地圖功能

## canvas 地圖

-   圖片置入

## 地圖區塊

-   SVG 區塊
-   城堡區塊

## 滑鼠圖示

-   圖示變換
-   進入區塊範圍移除圖示

## 區塊效果

-   點擊時顯示訊息
-   點擊地圖置中
-   Hover 效果

## 地圖效果

-   縮放(Press + Wheel)
    -   canvas min scale(0.6);max scale(1.7)
    -   svg width;height 會變換
-   拖拽
